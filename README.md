# 📌 BoxyChat

BoxyChat is a Discord-inspired chat platform built by BoxTech Studios for learning, experimentation, and future real-time communication features.

## 🌐 Goal
To build a modern chat system inspired by Discord with servers, channels, and messaging features.

## 🚧 Status
Pre-development (foundation setup only)

## 🧠 Tech (planned)
- HTML / CSS / JavaScript
- Node.js (future backend)
- WebSockets (real-time chat)
